# Plan Feature

I want to implement an RFC. Help me plan the implementation.

1. Ask me which RFC to implement (or let me specify the file path)
2. Read the RFC document
3. Read `docs/architecture.md` and `docs/tech_stack.md` for context
4. Read `tasks/lessons.md` for past corrections relevant to this type of work
5. Break the RFC down into concrete implementation steps
6. Write the implementation checklist into the RFC's `## Implementation Checklist` section
7. Suggest which step to start with and why

Do NOT start coding yet. This is planning only.
