# Create RFC

Read the skill in `.claude/skills/rfc-writer/SKILL.md` and follow its instructions to create a new RFC.

Interview me thoroughly before writing. Ask about the problem, the solution, edge cases, architecture impact, and testing approach. Don't ask obvious questions — dig into the hard parts I might not have considered.

Once the RFC is written, tell me the file path and suggest I review it before setting the status to `Accepted`.
